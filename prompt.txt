- Google Teachable Machine의 pose porject를 활용한 게임 웹사이트를 만들어라.
- html files: init.html, game.html, score.html
- init.html: {
    - 웹사이트에 접속했을 때 보이는 화면이다. 
    - 화면 중앙에 Teachable Machine에서 모델 업로드 시 추출되는 model URL을 입력하는 박스가 존재한다.
    - 박스 옆에 start button이 존재하고, 이를 클릭하면 game.html로 이동한다.
}
- game.html: {
    - game을 진행하는 main page이다.
    - 게임 방식: {
        - score는 0으로 시작한다. 남은 시간은 3 minutes으로 설정한다.
        - 모델의 label을 선택지로 제공하여 이 중 하나를 선택한다. 
        - 화면 중앙에 webcam이 실행되며, real-time으로 모델이 예측에 사용하는 점과 직선이 webcam 위에 표시된다. 
        - webcam 왼쪽 아래에는 사용자가 선택한 label이 표시되고, 그 오른쪽에 real-time으로 변동되는 model의 예측 확률에 따라 오른쪽으로 증가하는 bar가 존재한다.
        - threshold는 70%이며, 이를 넘어가면 점수를 1점 증가하고, 새로운 label을 선택하도록 한다.
        - 모든 label을 한 번씩만 선택될 수 있으며, 이전에 선택된 label은 다른 색으로 표시되고 선택할 수 없다.
        - 시간이 종료되거나 남은 label이 존재하지 않을 경우, game을 종료한다.
    }
    - 좌측 상단에는 score box가 존재한다.
    - score box 우측에 남은 시간을 표시하는 timer box가 존재한다.
    - game이 종료되면 남은 시간 10초마다 최종 점수에 0.1점씩 추가한다.
    - game이 종료되면 score.html로 이동한다.
}
- score.html: {
    - 화면의 중앙에 획득한 점수를 표시한다.
}
